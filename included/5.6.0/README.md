# ilyaztsv/cypress-included:5.6.0

Read [Run Cypress with a single Docker command][blog post url]

## Run tests

```shell
$ docker run -it -v $PWD:/e2e -w /e2e ilyaztsv/cypress-included:5.6.0
# runs Cypress tests from the current folder
```
